Apa itu Arduino ?

::marker Arduino adalah sebuah platform elektronik yang memungkinkan siapa pun, dari pemula hingga ahli, untuk merancang dan mengembangkan berbagai proyek elektronik interaktif. Dirancang dengan tujuan membuat pemrograman dan pengembangan perangkat keras menjadi lebih mudah diakses, Arduino telah menjadi alat yang sangat populer dalam bidang Internet of Things (IoT), robotika, seni interaktif, dan banyak lagi.

Arduino pertama kali dikembangkan oleh sebuah tim di Interaction Design Institute Ivrea (sekarang dikenal sebagai Interaction Design Institute di Domus Academy) di Italia pada tahun 2003. Awalnya, tujuan pengembangannya adalah untuk memberikan alat yang terjangkau bagi mahasiswa untuk belajar pemrograman dan elektronik. Namun, popularitasnya tumbuh pesat dan menjadi daya tarik di kalangan penggemar elektronik dan inovator.

Sketch pada Arduino

Dalam pemrograman Arduino, ada dua fungsi yang sangat penting: void setup() dan void loop(). Kedua fungsi ini digunakan untuk mengatur dan menjalankan program di papan Arduino. Berikut penjelasan tentang keduanya:

  • void setup()

    Fungsi setup() dijalankan sekali pada awal program. Ini digunakan untuk mengatur konfigurasi awal papan Arduino dan inisialisasi komponen yang diperlukan. 

                                
                                //Contoh
                                void setup() {
                                    pinMode(13, OUTPUT); // Mengatur pin 13 sebagai output
                                    Serial.begin(9600); // Memulai komunikasi serial
                                }

    Dalam contoh di atas, kita mengatur pin 13 sebagai output dan memulai komunikasi serial dengan kecepatan 9600 baud.

  • void loop()

    Fungsi loop() akan dijalankan secara terus-menerus setelah setup() selesai. Semua kode yang ada di dalam fungsi loop() akan diulang terus-menerus selama papan dinyalakan. 

                                
                                //contoh
                                void loop() {
                                digitalWrite(13, HIGH); // Menyalakan LED
                                delay(1000); // Tunggu 1 detik
                                digitalWrite(13, LOW); // Matikan LED
                                delay(1000); // Tunggu 1 detik
                              }

    Dalam contoh di atas, LED akan berkedip karena kode di dalam loop() terus dijalankan.

Variabel

Variabel dalam pemrograman Arduino bekerja dengan cara yang mirip dengan di pemrograman pada umumnya. Variabel digunakan untuk menyimpan data sementara yang akan digunakan atau diolah oleh program Anda. Berikut adalah beberapa hal penting tentang variabel dalam konteks pemrograman Arduino:

  • Deklarasi Variabel
    Anda harus mendeklarasikan variabel sebelum menggunakannya. Ini memberi tahu Arduino tentang jenis data dan nama variabel yang akan digunakan. 
                                
                                //Contoh
                                int ledPin = 13; // Deklarasi variabel ledPin dengan tipe int
                                float temperature; // Deklarasi variabel temperature dengan tipe float
  • Tipe Data
    Arduino mendukung beberapa tipe data dasar seperti int (bilangan bulat), float (bilangan desimal), char (karakter), dan bool (boolean).
  • Penugasan Nilai
    Anda bisa memberikan nilai ke variabel saat mendeklarasikannya atau kemudian dalam program menggunakan operator '='
                                
                                //contoh
                                int ledPin = 13; // Inisialisasi ledPin dengan nilai 13
                                temperature = 25.5; // Mengisi nilai ke variabel temperature
  • Penggunaan Variabel
    Variabel dapat digunakan dalam ekspresi matematika, operasi perbandingan, dan sebagainya. 
                                
                                //contoh
                                int x = 5;
                                int y = 10;
                                int sum = x + y; // Menyimpan hasil penjumlahan dalam variabel sum
  • Lingkup Variabel
    Variabel yang dideklarasikan di dalam fungsi setup() atau loop() hanya dapat diakses di dalam fungsi tersebut. Variabel yang dideklarasikan di luar fungsi (disebut variabel global) dapat diakses oleh semua fungsi dalam program.
  • Konstanta
    Anda juga dapat mendefinisikan konstanta yang nilainya tetap selama program berjalan dengan menggunakan 'const'. 
                                
                                //contoh
                                const int MAX_VALUE = 100; // Mendefinisikan konstanta MAX_VALUE
  • Menggunakan Variabel dalam String
    Anda dapat menggunakan variabel dalam string dengan menggunakan operator + atau menggunakan fungsi 'String()'. 
                                
                                //contoh
                                int sensorValue = analogRead(A0);
                                Serial.println("Nilai sensor: " + String(sensorValue)); // Menggunakan operator +

Variabel merupakan elemen penting dalam pemrograman Arduino dan memungkinkan Anda untuk menyimpan, memanipulasi, dan mengelola data dalam proyek Anda.

Digital Input/Output (I/O)

Digital Input/Output (I/O) adalah konsep dasar dalam pemrograman Arduino, yang memungkinkan Anda berinteraksi dengan sinyal digital, seperti saklar, tombol, LED, dan perangkat digital lainnya. Berikut adalah gambaran tentang digital I/O dalam Arduino:

  • Output Digital, melibatkan pengendalian perangkat yang memiliki dua kondisi: NYALA (HIGH) dan MATI (LOW)

    Fungsi-fungsi:

    pinMode(pin, mode): Mengatur mode pin sebagai INPUT atau OUTPUT.
    digitalWrite(pin, value): Menulis nilai digital ke pin. Nilai bisa HIGH (5V atau NYALA) atau LOW (0V atau MATI).
                                
                                //Contoh - LED Berkedip:
                                void setup() {
                                pinMode(13, OUTPUT); // Mengatur pin 13 sebagai output
                              }
                              
                              void loop() {
                                digitalWrite(13, HIGH); // Menyalakan LED
                                delay(1000); // Tunggu 1 detik
                                digitalWrite(13, LOW); // Matikan LED
                                delay(1000); // Tunggu 1 detik
                              }
  • Input Digital, Input digital melibatkan pembacaan status perangkat digital, seperti saklar atau tombol.

    Fungsi-fungsi:

    digitalRead(pin): Membaca status digital dari pin. Mengembalikan HIGH atau LOW.
                                
                                //Contoh - Input Tombol:
                                const int buttonPin = 2; // Tombol terhubung ke pin 2
                                int buttonState = 0;     // Variabel untuk menyimpan status tombol
                                    
                                void setup() {
                                    pinMode(buttonPin, INPUT); // Mengatur pin 2 sebagai input
                                }
                                    
                                void loop() {
                                    buttonState = digitalRead(buttonPin); // Membaca status tombol
                                    if (buttonState == HIGH) {
                                        // Tombol ditekan, lakukan sesuatu
                                    }
                                }
  • Resistor Pull-up dan Pull-down, Saat menggunakan tombol, umumnya menggunakan resistor pull-up atau pull-down untuk memastikan pembacaan yang stabil. Resistor pull-up menghubungkan pin input ke 5V, dan resistor pull-down menghubungkannya ke ground. Hal ini mencegah keadaan "mengambang" saat tombol tidak ditekan.

    Fungsi-fungsi

    digitalWrite(pin, value): Mengatur mode resistor pull-up atau pull-down. value bisa HIGH, LOW, atau INPUT_PULLUP.
                                
                                //Contoh - Tombol dengan Resistor Pull-up:
                                const int buttonPin = 2;
                                int buttonState = 0;

                                void setup() {
                                    pinMode(buttonPin, INPUT_PULLUP); // Menggunakan resistor pull-up internal
                                }

                                void loop() {
                                buttonState = digitalRead(buttonPin);
                                if (buttonState == LOW) {
                                    // Tombol ditekan, lakukan sesuatu
                                    }
                                }
Analog Input/Output (I/O)

Analog Input/Output (I/O) adalah konsep penting dalam pemrograman Arduino yang memungkinkan Anda untuk berinteraksi dengan dunia analog, seperti membaca sensor suhu, cahaya, atau kelembaban. Berikut adalah penjelasan tentang analog I/O pada Arduino:

  • Analog input melibatkan membaca nilai analog, yang dapat bervariasi dalam rentang tertentu, misalnya suhu atau intensitas cahaya.

    Fungsi-fungsi:

    analogRead(pin): Membaca nilai analog dari pin. Mengembalikan nilai antara 0 dan 1023.
                                
                                //Contoh - Membaca Sensor Suhu LM35:
                                const int analogPin = A0; // Sensor suhu terhubung ke pin A0

                                void setup() {
                                  Serial.begin(9600); // Memulai komunikasi serial
                                }
                                
                                void loop() {
                                  int sensorValue = analogRead(analogPin); // Membaca nilai sensor
                                  float temperature = (sensorValue * 5.0 / 1023.0 - 0.5) * 100.0; // Menghitung suhu
                                  Serial.print("Suhu: ");
                                  Serial.print(temperature);
                                  Serial.println(" Â°C");
                                  delay(1000); // Tunggu 1 detik
                                }
  • Analog output melibatkan mengeluarkan sinyal analog yang dapat bervariasi dalam rentang tertentu, seperti mengontrol kecepatan motor DC.

    Fungsi-fungsi:

    analogWrite(pin, value): Mengeluarkan sinyal PWM (Pulse Width Modulation) pada pin. Nilai value antara 0 (mati) dan 255 (maksimum).
                                
                                //Contoh - Kontrol Kecerahan LED:
                                const int ledPin = 9; // LED terhubung ke pin 9

                                void setup() {
                                  pinMode(ledPin, OUTPUT); // Mengatur pin sebagai output
                                }
                                
                                void loop() {
                                  int brightness = analogRead(A0); // Membaca nilai sensor (misal: LDR)
                                  analogWrite(ledPin, brightness / 4); // Mengatur kecerahan LED
                                }
